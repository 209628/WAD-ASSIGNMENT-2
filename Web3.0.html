WEB 3.0

Definition:

- It is the third generation of the World Wide Web that began emerging around 2010 and is still developing today.

- It is often referred to as the "Read Write Execute Web" or the "Semantic Web" because it focuses on making the internet more intelligent, decentralized and user centered.

- Unlike Web 2.0 where large companies such as Google, Facebook and Amazon control most of the data and content Web 3.0 aims to give users more control over their own data and digital identities.

- It uses artificial intelligence, machine learning and blockchain technology to create a smarter and more personalized web experience.

Characteristics of Web 3.0:

  1. It is decentralized
  2. It is semantic
  3. It supports artificial intelligence and machine learning
  4. It incorporates blockchain technology which provides a secure and transparent way of storing and transferring data.

* Well known examples of Web 3.0 technologies and platforms include cryptocurrency such as Bitcoin and Ethereum, decentralized applications known as DApps, non fungible tokens known as NFTs and decentralized finance platforms known as DeFi.
