WEB BROWSER

Definition:

- A software application used to locate retrieve and display content on the World Wide Web.

- A piece of software that runs on your computer and allows you to view web pages.


Steps for how a Web Browser works:

  1. You type a URL in your browser e.g; www.google.com
  2. Your computer asks a DNS Server to translate the URL into an IP Address like 142.250.80.46
  3. Your browser sends an HTTP/HTTPS request to the Web Server at that IP Address
  4. The Web Server receives the request; it sends back the HTML,CSS and images that make up the webpage
  5. Your browser reads the HTML and CSS; it renders everything into the visual webpage you see on screen.

N/B: Security is also an important function of a web browser. Modern browsers warn users about unsafe websites, block harmful downloads and support HTTPS connections which encrypt data being sent between the browser and the web server. This helps protect users from hackers and online threats.
