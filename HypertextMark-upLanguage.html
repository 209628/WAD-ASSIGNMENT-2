HYPERTEXT MARK-UP LANGUAGE

Definition:

- It is the standard language used to create and structure web content.

- It is the backbone of every webpage and it tells the browser how to display text, images, links, videos and other content on a webpage.

- It uses a system of tags to mark up text so browsers know how to display it.

Basic structure of HTML:

  1. The !DOCTYPE html declaration tells the browser that the document is written in HTML5.
  2. The html tag is the root element that wraps all the content on the page.
  3. The head section contains information about the page such as the title and metadata that is not displayed on the page itself.
  4. The body section contains all the content that is displayed on the webpage such as headings, paragraphs, images and links.

How it works:

  • Tags are written inside angle brackets(< >) for headings. E.g; h1 for headings, p for paragraphs, img for images and a for hyperlinks.
  • Most HTML tags come in pairs with an opening tag and a closing tag.
  • The content that goes between the opening and the closing tags is what gets displayed on the webpage.
  • The browser reads the HTML code from top to bottom and renders it into visual webpage that the user sees on their screen.
