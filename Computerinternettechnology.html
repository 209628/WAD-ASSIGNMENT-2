COMPUTER INTERNET TECHNOLOGY

The internet uses TCP/IP to connect devices, while the World Wide Web lets users access information through browsers and servers using URLs, DNS, and protocols like HTTP and FTP, with content built in HTML. It includes private networks such as intranets & extranets and has evolved from static Web 1.0 to interactive Web 2.0, intelligent Web 3.0, and AI-driven Web 4.0.
