URI

Definition:

- It is a string of characters that uniquely identifies a particular resource on the internet or on a network.

- A URI can identify a resource either by its location, by its name or by both at the same time.

- Every URL and every URN is a type of URI making URI the parent term that covers all methods of identifying resources on the internet.

Structure of a URI:

- A URI is made up of several components that work together to identify a resource.

  • The first component is the scheme which specifies the protocol or method used to access the resource such as HTTP, HTTPS, FTP or mailto
  • The second component is the authority which identifies the host or server where the resource is located such as www.example.com
  • The third component is the path which specifies the exact location of the resource on the server such as /images/photo.jpg
  • Some URIs also include a query string which passes additional parameters to the server and a fragment which points to a specific section within a resource.
