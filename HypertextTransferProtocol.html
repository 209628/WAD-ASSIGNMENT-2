HYPERTEXT TRANSFER PROTOCOL

Definition:

- It is an application-layer protocol(set of rules) that govern how data is transferred between a Web Browser and Web Server over the internet.

- It defines how messages are structured and exchanged on the web.

- Every time you visit a website your browser uses HTTP to request the webpage from the server and the server uses HTTP to send the webpage back to your browser.

How it works:

  • The browser sends a request to the web server using a method such as GET, POST, PUT or DELETE
  • The web server receives the request and reads the method to understand what action needs to be performed
  • The server processes the request by locating the requested resource or performing the requested action such as retrieving or updating data
  • The server responds by sending back a status code and the requested content to the browser
  • The browser receives the response, reads the status code and displays the content to the user on their screen.
