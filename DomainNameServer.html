DOMAIN NAME SERVER

Definition:

- It is a system that translates domain names into IP addesses automatically when you type a URL.

- DNS is the internet's phone book.

How it works:

  1. You type a website
  2. Your device sends a request to a DNS server
  3. The DNS server returns the IP address
  4. Your device connects to that

N/B: If the DNS server does not recognize the domain name it will pass the request to a higher level DNS server until the correct IP address is found or an error is returned.

* DNS can be attacked by hackers through a method called DNS Spoofing where fake DNS responses are used to redirect users to malicious websites.

- This is why secure DNS protocols such as DNSSEC were developed to protect users

* DNS is often referred to as the "Phone Book of the Internet" because just like a phone book converts a persons name into a phone number, DNS converts a domain name into an IP address.

Did you know: Every time you visit a website your browser performs a DNS lookup in the background without you even noticing. This entire process happens in milliseconds.
