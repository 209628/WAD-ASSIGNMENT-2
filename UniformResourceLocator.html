UNIFORM RESOURCE LOCATOR(URL)

Definition:

- An identifier for the location of a document on a Web site.

- Every source on the internet has its own unique URL which acts like a home address telling your browser exactly where to find it.

Structure of a URL:


A URL is made up of several parts that work together to locate a resource:

  1. Protocol - The first part which is usually HTTP or HTTPS and it tells the browser how to communicate with the server.
  2. Domain Name - The second part such as www.google.com which identifies the website.
  3. Path - The third part which points to a specific page or file on that website such as /search or /about.
