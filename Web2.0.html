WEB 2.0

Definition:

- It is the second generation of the World Wide Web that emerged around 2004 and is still in use today.

- It is often referred to as the "Read Write Web" because unlike Web 1.0 where users could only read content Web 2.0 allows users to both read and create their own content.

- Web 2.0 introduced a new era of interactivity, collaboration and user participation on the internet. It transformed the web from a one way information source into a dynamic platform where users could interact, share and contribute content freely.

Characteristics of Web 2.0:

  1. It is dynamic and interactive
  2. Supports user-generated content
  3. Introoduced social networking, blogging, wikis, online shopping and video sharing platforms
  4. It aslo introduced rich user interfaces with better design, animations and interactive features.

* Well known examples of Web 2.0 platforma include Facebook, YouTube, Twitter, Wikipedia, Instagram and WordPress.
